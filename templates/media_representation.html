Media Representation: whose stories does the media choose to represent?

In this section, we will be looking at which locations the top national news outlets reference most when providing coverage on ICE activity. We find that media coverage is oversaturated in certain states, such as Minneapolis and Illinois, compared to the actual number of arrests occurring in that state.

How to interact

Below, you can select between plotting different datasets, over different periods of time.

Dataset options: The datasets include different filters that change the news articles considered covering "ICE activity". See the data collection section for more information on these differences.

Time range: We include a time selector because these trends change over time, for example, if selecting the start date to be 12/01/25, Minneapolis has a much higher saturation than the default. The default start date is 1/23/25, as this is when Trump began making policy changes regarding the DHS and ICE enforcement policy. The end date is 4/25/26, which is when we most recently updated this data collection.

Data Collection and Limitations

Graphic: article mentions by state

Our article data was sourced from GDELT, using their thematic filtering tags, from the following ten sources: New York Times, Fox News, CNN, USA Today, Washington Post, Wall Street Journal, AP News, NBC News, NPR, and Breitbart.

The criterion used to filter articles that mention ICE spottings is very crude, since GDELT does not have a built-in theme for ICE, only the Department of Homeland Security. Thus, the captured articles capture a bit more than just ICE actions, such as administrative or budgeting concerns with the Department of Homeland Security. To ameliorate this effect, we tried to filter for themes such as arrests, migration/immigration, violence, and unrest.

Users are able to see the impact of using different dataset options for the geographical plots on the resulting conclusions. Interestingly, there is little change between filters on observable patterns. In our future steps, we would like to do a more systematic review of the themes included in our dataset to create a better filter, without baking in our assumptions about what ICE activity looks like. \n

The visualization is created by calculating the number of times a location is mentioned in articles within the dataset, using GDELT’s auto-detected location tags. We acknowledge that not all times a location is mentioned in an article, will it necessarily be used as a locational description of where ICE activity is occurring. However, this is not about logic, but about human psychology and the impact media representations cause on where people perceive ICE activity to be occurring, and at what rate. The more times that a location is mentioned in the same context as ICE, the more likely a user is to create an association between the two. This is not a perfect method, but there is no method using our dataset to see the relationship between each location mentioned in the text to the themes in the text, so we must settle for this imperfect methodology.

Other notes

The granularity of our observations at the state level is because our data has mixed location tags at the state level and the city/county level, from both the Deportation Data Project and the GDELT database. In order to not falsely overrepresent smaller locations because of a lack of more fine-grained data, we stay at state level representations.
